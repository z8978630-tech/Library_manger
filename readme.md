
# پروژه کلاس Car

 هدف
این پروژه برای تمرین مفاهیم پایه شیءگرایی در پایتون شامل کلاس، شی، متد، نوشته شده است.

 مفاهیم به کار رفته
- تعریف کلاس (`class Car`)
- متد سازنده (`__init__`)
- ویژگی‌های شیء (`self.model`, `self.color`)
- متدها: `info()`, `change_color()`, `get_info()`, `is_same_color()`
